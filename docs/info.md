## How to test
The first two pins are used for operation 
00 ------ Addition 
01 ------ Multiplication 
11 ------ Subtraction

pin 3 --------- A2
pin 4 --------- A1
pin 5 --------- A0 
pin 6 --------- B2
pin 7 --------- B1
pin 8 --------- B0

S2,S1,S0 which is given as input to 7 segment display
